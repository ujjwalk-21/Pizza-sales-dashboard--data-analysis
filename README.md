# Pizza-sales-dashboard--data-analysis 
#The goal of this project is to build an interactive Power BI dashboard that consolidates pizza sales data, tracks key metrics (such as total sales, orders, and pizzas sold), and provides visual insights to support data-driven decisions, optimize operations, and boost revenue.
## project objective
The objective of this project is to analyze pizza sales data and create an interactive Power BI dashboard that provides insights into overall sales performance, product popularity, peak order times, seasonal trends, and regional or salesperson performance. The dashboard aims to enable data-driven decisions for optimizing operations, improving sales, and enhancing customer satisfaction.
